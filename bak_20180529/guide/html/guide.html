viewport 선언

			<meta name="viewport" content="width=device-width, initial-scale=1, user-scalable=yes, minimum-scale=1, maximum-scale=1" />
속성 설명 값
width viewport의 가로크기를 지정 device-width
initial-scale 페이지가 최초로 로드될 때 사용할 기본 줌인/줌아웃 값, 1은 100%를 의미 1(기본값) ~ 10
user-scalable 사용자에게 줌인/줌아웃을 허용할지 여부를 설정 yes(기본값) / no
minimum-scale 최소 줌아웃 배율로 지정한 배율 이하로 축소할 수 없음 0 ~ 10 / 기본값 (0.25)
maximum-scale 최대 줌아웃 배율로 지정한 배율 이하로 확대할 수 없음 0 ~ 10 / 기본값 (1.6)
